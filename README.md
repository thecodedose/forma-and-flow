## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Update the services/contact forms

[Create free forms using Tally](https://try.tally.so/codedose) and update the form links in the Services and Newsletter section.
